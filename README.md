# Docker Mini Project

A simple Node.js + MongoDB application containerized using Docker and Docker Compose.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Docker
- Docker Compose

## Project Structure


docker-mini-project/
│── app/
│── docker-compose.yml


## Run the Project


docker compose up --build


Open:


http://localhost:3000


## Features

- Multi-container application
- Docker Compose
- Docker Networking
- Docker Volumes
- MongoDB Integration
